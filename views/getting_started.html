Guide de démarrage

Accueil Parent

Bienvenue sur votre console d'administration GHome !

Ce petit tutoriel vous guidera à travers la prise en main rapide de votre serveur GHome.

Page d'accueil

Sur votre page d'accueil, vous pouvez retrouver les différentes fonctionalités de votre console d'administration GHome. Parmi celles-ci :

Gestion des équipements
Vous permet de voir l'état courant de vos équipement et d'intéragir avec eux le cas échéant. C'est également là que vous pouvez ajouter un nouvel équipement.
Statistiques
Vous permet de visualiser l'évolution de différentes données de votre GHome au cours du temps. Comme la température intérieure et la consommation électrique.
Mode espion
Vous permet de paramétrer l'activation du Mode espion ainsi que d'en visualiser l'historique (logs).
Planificateur
Fonctionnalité principale de votre GHome, le planificateur vous permet d'effectuer toutes sortes d'actions en fonction de différents paramètres ainsi que lors d'évènements en provenance des capteurs enregistrés dans votre GHome.
Véritable boîte à outils, il vous permettra de rendre votre maison plus intelligente encore qu'elle ne l'est avec la configuration par défaut de nos génies d'ingénieurs.
Seuils
Vous permet de gérer les différents "seuils" associés aux différents types de capteurs rattachés à votre GHome.
Ces seuils sont ensuite utilisés par le planificateur pour prendre ces décisions.
Guide de démarrage
La page sur laquelle vous vous trouvez en ce moment-même.

Ajout d'un nouvel équipement

Afin d'ajouter un nouvel équipement, sur la page d'accueil, choisissez l'option Gestion des équipements.
Appuyez ensuite sur Nouvel équipement en haut de la page de gestion des équipements. Il ne vous reste
plus qu'a entrer les bonne informations selon ce qui vous est demandé.
Les identifiants doivent être entrés en décimal.

Rattachement d'un équipement sous Android à votre serveur GHome

Installez l'application via le ficher .apk fourni (si non visible sur le périphérique, activez "sources inconnues")
Lancez l'application installée ci-dessus
Saisissez l'adresse IP, le login et le mot de passe vous permettant de vous connecter au serveur
Appuyez sur Valider
Attendez entre 10s et 3 minutes que la synchronisation entre votre appareil et le serveur s'établisse
Une notification "Vous êtes connecté à votre serveur GHome" devrait apparaitre, le périphérique est alors connecté au serveur
Vous pouvez désormais utiliser notre application sur votre système android !

Ajout d'une nouvelle tâche planifiée

Afin d'ajouter une novuelle tâche planifiée, sur la page d'accueil, choisissez l'option Planificateur.
Appuyez ensuite sur Nouvelle tâche. Il ne vous reste qu'à nommer votre nouvelle tâche et à définir l'action
à éffectuer lorsqu'elle sera lancé par l'évènement que vous aurez choisit. Si vous le souhaitez vous pouvez également
ajouter des conditions afin d'afiner votre tâche pour la lancer uniquement lorsque VOUS le souhaitez.

Ajout d'un seuil

Afin d'ajouter un nouveau seuil, sur la page d'accuel, choisissez l'option Seuil.
Appuyez ensuite sur Nouveau seuil.Il ne vous reste ensuite qu'à nommer votre nouveau seuil ainsi que de lui
attribuer la valeur que vous souhaitez. Il faut enfin définir les capteurs concernés par ce seuil. Ils seront ainsi capable
de lancer un évènement quand il dépasseront votre nouveau seuil.

Modification d'un seuil

Afin de modifier un seuil, sur la page d'accuel, choisissez l'option Seuil.
Appuyez ensuite sur le seuil que vous souhaitez éditer.Il ne vous reste ensuite qu'à éditer votre seuil.

Visualisation des statistiques

Afin de visualiser les statistiques, sur la page d'accueil, choisissez l'option Statistiques.
Il vous suffit ensuite de sélectionner la date de début ainsi que la date de fin de vos statistiques
et de sélectionner le type d'information que vous désirez voir. Cliquer enfin sur valider afin de
visualiser vos statistiques.
